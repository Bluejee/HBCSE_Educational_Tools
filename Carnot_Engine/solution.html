Solving the equation

  • We have equation of efficiency as

    • $$\eta = 1 - \frac{T_C}{T_H}$$

    We can calculate the efficiency for a given carnot engine by using this equation and analyze optimum conditions for maximum efficiency.

  • For example let the temprature of the source and sink be the boiling and freezing point of water. Then we have,

(Temprature in kelvin), $$T_H = 373,~~T_C = 273$$ This gives the efficiency of the engine to be, $$\eta = 1 - \frac{T_C}{T_H} = 0.27$$

We can see that even for a most ideal engine, the efficiency we can achive using real world tempreatures is very low.
